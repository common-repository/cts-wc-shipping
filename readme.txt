== Description ==

= Introduction =
Shipping plugin for WooCommerce is template plugin which allow to show some shipping rate on Woocommerce cart/checkout page. The plugin is designed to integrate any other API Like UPS, FedEx, DHL etc.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress admin.
